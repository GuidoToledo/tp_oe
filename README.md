# TP Organización Empresarial
## Integrantes
- Guido Toledo
## Escenario elegido
Escenario B - Análisis de Ventas
## Descripción
El proyecto analiza un conjunto de datos usando Python con una tabla de ventas extraida de:
https://toledoguido.atlassian.net?continue=https%3A%2F%2Ftoledoguido.atlassian.net%2Fwelcome%2Fsoftware&atlOrigin=eyJpIjoiNzM2ZGFjZTY1Y2E3NDA0NThhMDQyZGIxY2FlMGFkOWQiLCJwIjoiaiJ9
como recomienda el TP, uso Pandas y Matplotlib para trabajar con tablas y hacer graficos respectivamente.
## El script permite:
- calcular ventas totales
- obtener ventas por mes
- calcular promedios mensuales
- generar gráficos de ventas
## Estructura del proyecto
- `/datos` - datos utilizados
- `/scripts` - codigo
- `/resultados` - graficos y archivos generados
